PageRank

As mentioned in the introduction of this website, PageRank is what Google uses to rank pages showing up in the search results. If you have installed the Google toolbar in your browser, you can find PageRank there. The Toolbar PageRank only goes up from zero to ten, but it seems to be a logarithmic scale. However, because pages are added and removed every hour, it is not that simple. PageRank will sometimes even need to guess, for example for newly added pages.

To simply put it: PageRank works by counting votes. A link to a page counts as a vote. This means that if there are no links, there are no votes for that page. The results do not subtract votes in any way. So now that we know that the PageRank of each page depends on how many times it has been ‘voted’ on, don’t we know how it works? Unfortunately, no. Before being able to know whether or not the vote to a page A is valid, we need to know the PageRanks of the page B that linked to page A. Since pages can link to each other and create circles, how are we ever going to be able to calculate the PageRanks?

Because there is absolutely no way of explaining how PageRank works easily and in other words than the website that has helped us understand, we will link you to that website. Here you go, no thanks needed.
